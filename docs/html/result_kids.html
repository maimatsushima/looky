あなたはこんな人かも！

給食は残さない、お母さんに言われた門限は守る（夏は18時まで、冬は17時までね）、明日の時間割は前日までに準備する。例えばそんな小学生だったかも。みんなと仲良く楽しく過ごせたらいいなという平和主義のもと、グループで決めたルールは絶対守ります。真面目で純粋な気持ちのまま大人になっていくあなた。もちろんとても良いことなのですが、たまには友達がまだ誰もやっていないような何か新しいことにチャレンジしてみることも大事かもしれませんよ。

あなたに特に合うお仕事は

エンジニア

一生懸命で真面目なあなたは誰かを支えたり、人のために頑張ることができます。また、毎日アンテナをはってコツコツ技術を習得することもできるので、専門スキルを使ってサービスを支えるようなエンジニアが向いています。みんなを協力的に支えようと頑張るあまり、少し優柔不断になってしまうことも。たまにはリーダーや先輩のような先人を真似して自分で物事を決めたり最後までやり抜こうとしてみても良いかもしれませんね。
