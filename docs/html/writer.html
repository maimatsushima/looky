ライター

制作・編集・校正の仕事は書籍や雑誌など出版物全般の企画、作家やクリエイターへの仕事依頼、原稿の整理・チェック・公正、誌面構成、取材、DTP指示、印刷管理など出版物が完成するまでのあらゆる段階に関わる幅広い仕事です。時には自分で原稿を執筆することもあり、自分が携わった企画が印刷物になって多くの人に読まれるという喜びがあります。

書き終わり次第、先方へチェック依頼。

13:00 ランチ

自分で家でランチを作って、

14:00 リサーチ

新しい記事の執筆のために、情報をリサーチします。
必ずしも自分が知っている情報だけで
記事作成ができないので、リサーチ能力も大事です。

16:00 記事作成

リサーチをした内容で新しい記事を作成。
この記事を書き上げたら今日のお仕事終了…！

18:00 帰宅

ゆっくり切り替えて過ごします。
