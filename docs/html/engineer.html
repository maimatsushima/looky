12:00 社内ミーティング

チーム内のミーティングで進捗を共有。

14:00 小腹休憩

小腹がすいたので、少しご飯をつまむ。
休憩中は今はまっているアニメを流しながら、

14:30 集中タイム

コーディングをゴリゴリ進めます。
アニメの音声をBGMにしながら。
意外と集中できるもんです。

集中して頭が疲れたので、帰宅。
